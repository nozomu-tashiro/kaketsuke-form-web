# 販売店情報印字フォーマット実装完了

## 実装内容

### 印字レイアウト

```
┌─────────────────────────────┬──────────────────────┐
│ 販売店名（電話番号）        │  販売店コード        │
│ いえらぶ不動産販売株式会社  │  13-00-11223366-000  │
│ 03-1234-5678               │                      │
├─────────────────────────────┼──────────────────────┤
│                             │  担当者名            │
│                             │  いえらぶ太郎        │
└─────────────────────────────┴──────────────────────┘
```

### 実装詳細

#### 1. フォントサイズ
- **15pt (xlarge)** を使用
- 全ての販売店情報フィールドに適用

#### 2. 配置ルール
- **左寄せ**: すべてのテキストを左寄せで配置
- **折り返し対応**: 長いテキストは自動で複数行に折り返し
- **行間**: 18pt

#### 3. フィールド配置

**上段左側ボックス**:
- 販売店名: X=130, Y=130
- 電話番号: 販売店名の下に続けて表示
- 横幅: 約220pt

**上段右側ボックス**:
- 販売店コード: X=385, Y=130
- 横幅: 約180pt

**下段右側ボックス**:
- 担当者名: X=385, Y=95 (上段から35pt下)
- 横幅: 約180pt

## データ構造

### フロントエンド (ApplicationForm.js)
```javascript
agentInfo: {
  name: '',              // 販売店名
  phone: '',             // 電話番号
  code: '',              // 販売店コード
  representativeName: '' // 担当者名
}
```

### バックエンド (pdfGeneratorV5.js)
```javascript
agentInfo.name              // 販売店名
agentInfo.phone             // 電話番号
agentInfo.code              // 販売店コード
agentInfo.representativeName // 担当者名
```
