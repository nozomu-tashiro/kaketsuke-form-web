## 📋 プロジェクト概要

不動産関連の駆付けサービスの入会申込手続きをデジタル化するWebアプリケーションです。販売代理店の担当者がWebフォームに必要な情報を入力し、選択した商品とオプションに応じた申込書をPDF形式で生成・出力します。

## 🚀 起動方法

### 開発環境

```bash
# 1. 依存パッケージのインストール
cd backend && npm install
cd ../frontend && npm install

# 2. バックエンド起動（ターミナル1）
cd backend
npm run dev

# 3. フロントエンド起動（ターミナル2）
cd frontend
npm start
```

### アクセスURL
- **フロントエンド**: http://localhost:3000
- **バックエンドAPI**: http://localhost:5000/api

### 本番環境

```bash
# 1. フロントエンドのビルド
cd frontend
npm run build

# 2. バックエンド起動
cd backend
NODE_ENV=production node server.js
```

## 🐛 既知の問題

### 軽微な問題
1. **PDFの日本語フォント**
   - PDFKitは標準フォントで日本語に制限あり
   - 解決策: カスタムフォント（IPAフォント等）の追加が必要

2. **入力データの永続化**
   - 現在、リロードするとデータが消失
   - 解決策: ローカルストレージまたはデータベース実装

### 将来的な改善点
- PDF生成速度の最適化
- より詳細なエラーメッセージ
- アクセシビリティの向上
